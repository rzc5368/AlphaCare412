In this AplphaCare application I have implemented my iterator and search UI. Within this file I follow the Model View Control(MVC) pattern. 

My Controller class is called "hcControl". It contains the classes to be reached when instantiated from any class. 

My model class is called "hcModel". It contains the all the business logic of the iteration backend. 

My view class is called "hcView". It contains the all the user interface implementation required.
